010LoserTree
============

1) 입력 데이터를 읽어 오름차순으로 정리하시오. 입력 데이터는 다음과 같은 형식을 갖는다.

%cat input.txt
n	 k			// n = total # of nodes, where n = 40
				// k = # of runs, where k = 8
15   16   20   30    35	// Each run is a nondecreasing order
    20   38   43   45    53
    20   30   36   41    44
    15   25   28   29    32    
    15   30   40   50    60
    11   16   26   36    46
    71   72   83   91    99
    18   48   68   79    80




①  입력 데이터를 읽어 loser tree를 구성하시오. 처음 구성된 loser tree의 노드 4, 5, 6, 7의 값을 차례대로 출력하시오.




②  위의 ①에서 구성한 loser tree를 이용하여 오름차순으로 정리하시오
